# Convocation Gown System

## Introduction
The convocation gown system is designed to streamline the process of gown rental and return for graduates.

## Features
- Easy gown reservation process
- User-friendly interface
- Real-time availability tracking

## Installation
To install the Convocation Gown System, follow these steps:
1. Clone the repository:
   ```
   git clone https://github.com/raviamrav/convocation-gown-system.git
   ```
2. Navigate to the project directory:
   ```
   cd convocation-gown-system
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage
Once installed, you can start the application by running:
```bash
npm start
```
